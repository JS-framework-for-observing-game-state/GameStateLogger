# JS Framework for Observing Game State implemented in TicTacToe
In this example, [Canan Korkut's JavaScript TicTacToe implementation](https://github.com/canankorkut/TicTacToe) has been used as a base for the GameStateLogger.   
   
**Set-up instructions:**  
To install required dependencies, in your terminal run `npm i` from `/gamestatelogger/server`.   
   
**Run instructions:**   
To start the TicTacToe game in a browser, run it with your prefered HTML file run method.   
For Visual Studio Code we recommend the Live Server extension.     
   
To run the server-side of GameStateLogger, to receiving data from the game and create JSON files:   
From `/gamestatelogger/server` run:
```code
  node server.js
```
   
Logged events can be seen in the browser terminal, as well as the JSON file(s) logged in `gamestatelogger/server/JSONfiles`.
